Open Command Folder

Description

"Open Command Folder" is a Visual Studio Code Extension that opens a command prompt window from the folder that the currently opened file is in.

Video and Screenshots
Video Timestamps

- Instructions

- Start of play through. Top right of screen shows how much coins you currently have. Top left of screen shows the number of lives you have left. Underneath on the left side is the upgrades. The name, current cost, and a progress bar showing how much you have purchased each upgrade is shown on each upgrade button. When hovering over the upgrade buttons, a description of what the upgrade does is shown.

- Start of a bug attack. You must click all of the bugs before they disappear and steal your coins/lives.
